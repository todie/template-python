## Overview

 is an opinionated, production-ready Python project scaffold.
Clone it, rename it, ship it — everything is wired up from day one.

| What | Tool |
|------|------|
| Package manager | [uv](https://docs.astral.sh/uv/) |
| Linter & formatter | [ruff](https://docs.astral.sh/ruff/) |
| Type checker | [pyright](https://github.com/microsoft/pyright) (strict mode) |
| Test runner | [pytest](https://pytest.org/) with asyncio + coverage |
| Git hooks | [pre-commit](https://pre-commit.com/) |
| Security | [pip-audit](https://github.com/pypa/pip-audit) |
| CI/CD | GitHub Actions + release-please |
| Container | Multi-stage Dockerfile with non-root runtime |

## Quick Start

1. Click **[Use this template](https://github.com/todie/template-python/generate)** above.
2. Clone your new repo and run .
3. Find-and-replace  /  with your package name.
4. Update  metadata.
